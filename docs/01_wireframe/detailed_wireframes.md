# 詳細ワイヤーフレーム (Detailed Wireframes) - enna

本ドキュメントは、各ページの実装に必要なコンテンツ、レイアウト、UIパーツを詳細に定義します。
**前提**: Mobile First設計（基本は1カラム縦積み）、PCは適宜2カラム等へ拡張。

---

## 1. Top Page (`index.astro`)
サイトの顔。安心感を与え、事前面談への導線（CTA）を確実に踏ませる。

### 1-1. Hero Section (FV)
*   **Layout**: 画像背景 or 画像横並び（SPは画像上、テキスト下）
*   **Visual**: 窓辺で赤ちゃんを抱く保育士、またはあたたかい日差しのイメージ（AI生成）。
*   **Content**:
    *   **Catch Copy (Main)**: 「子どもに寄り添う、オンリーワンの保育」
    *   **Sub Copy**: 「ママと赤ちゃんを包み込む、プロフェッショナルの温もり」
    *   **Tagline**: 産後ケア施設勤務 × 保育士歴13年
    *   **CTA Button**: 「まずは事前面談(オンライン)へ」 (Primary Color, Pill Shape, Floating Shadow)
        *   ※SP時は画面下部に追従バナーとして表示する検討もあり

### 1-2. Introduction (共感)
*   **Heading**: 「ひとりで頑張りすぎていませんか？」
*   **Body**: 
    *   「初めての育児で不安...」「自分の時間が少しだけ欲しい...」
    *   そんなママの気持ちに寄り添い、プロとしてサポートするメッセージ。

### 1-3. Features (選ばれる3つの理由)
*   **Heading**: 「enna が選ばれる理由」
*   **Layout**: SP: Vertical Stack / PC: 3 Columns
*   **Item 1: 赤ちゃん期のプロ**
    *   Icon: 👶 (Baby icon)
    *   Title: 「新生児〜0歳児の専門性」
    *   Desc: 産後ケア施設勤務・保育士歴13年。退院直後から安心してお任せください。
*   **Item 2: 独自の知育遊び**
    *   Icon: 🎨 (Art/Palette icon)
    *   Title: 「感性を育むアクティビティ」
    *   Desc: 小麦粉粘土や手形アートなど、成長に残る体験を提供します。
*   **Item 3: 心のケア**
    *   Icon: 💗 (Heart icon)
    *   Title: 「ママへの傾聴と共感」
    *   Desc: 子育て心理アドバイザーとして、育児の悩みもしっかり受け止めます。

### 1-4. Service Preview
*   **Heading**: 「サービス内容」
*   **Content**: 簡単なメニュー紹介（シッティング、産後ケア、送迎など）
*   **Link**: 「詳しく見る >」 (Secondary Button)

### 1-5. News (microCMS)
*   **Heading**: 「お知らせ」
*   **List**: 最新3件を表示 (Date + Title)。クリックで詳細モーダル or 別ページへ。

### 1-6. Reviews (実績)
*   **Heading**: 「お客様の声」
*   **Stats**: 「実績 300回以上 / 評価 ★5.0」を大きく数字で見せる。
*   **Review Cards** (Carousel):
    *   ユーザー属性（30代 ママ）, コメント抜粋, ★★★★★

### 1-7. Bottom CTA
*   **Text**: 「まずはオンラインでお話ししませんか？」
*   **Button**: 「お問い合わせ・面談予約」

## 2. Profile Page (`profile.astro`)
「どんな人が来るの？」という不安を払拭し、人柄への信頼を醸成する。

### 2-1. Profile Hero
*   **Element**: プロフィール写真（円形または角丸、笑顔）
*   **Name**: 名前（アルファベット + 漢字）
*   **Title**: 産後ケアシッター / 保育士

### 2-2. Message (想い)
*   **Layout**: テキストメイン、行間ゆったり。
*   **Content**: 
    *   保育士を目指したきっかけ
    *   産後ケアへの想い
    *   「傾聴」を大切にしている理由

### 2-3. History & Certifications
*   **History (年表スタイル)**
    *   認可保育園 勤務
    *   インドネシア日系幼稚園 勤務（多様な文化適応）
    *   産後ケア施設 勤務（現職）
*   **Certifications (リスト表示)**
    *   保育士 / 幼稚園教諭
    *   子育て心理アドバイザー
    *   わらべうたベビーマッサージ講師
    *   市民救命士 / 安全サポート研修受講済

## 3. Service Page (`service.astro`)
料金体系を明確にし、納得感を高める。

### 3-1. Basic Plan (Pricing Table)
*   **Item**: 基本シッティング（1時間あたり）
*   **Price**: ¥X,XXX (内税/外税 表記確認)
*   **Condition**: 最低3時間〜 / 対象年齢 0歳〜

### 3-2. Options
*   **Time**: 土日祝 (+20%) / 早朝・夜間 (要確認)
*   **Care**: 病児保育 (+¥500/h)
*   **Travel**: 交通費（実費 / 宮前区・青葉区の一部は自転車可）

### 3-3. Activities (独自価値)
*   写真付きで視覚的にアピール。
*   **Item**: 小麦粉粘土 (+¥500)
    *   Text: 口に入れても安全な素材で、五感を刺激します。
*   **Item**: 手形足形アート (無料/有料要確認)
    *   Text: 成長の記録を可愛く残します。

### 3-4. Area (対応エリア)
*   **Map/Text**:
    *   拠点: 神奈川県川崎市宮前区
    *   詳細: 東急田園都市線沿線（たまプラーザ、鷺沼など）、東京一部
    *   条件: 自転車圏外は3時間〜利用可

## 4. Flow & FAQ Page (`flow.astro`)
利用のハードルを下げ、疑問を先回りして解消する。

### 4-1. Usage Flow (Step UI)
*   **Step 1: お問い合わせ** (フォームまたLINE)
*   **Step 2: 事前面談 (オンライン)** (必須, ¥1,250/30分)
    *   ※Zoom等を使用。相性確認とヒアリング。
*   **Step 3: ご予約**
*   **Step 4: シッティング当日**

### 4-2. FAQ (Accordion UI)
*   **Q**: 病児保育はどこまで対応できますか？
    *   **A**: 37.5度以上でも可ですが、インフル/コロナ/胃腸炎など感染症は不可です。診断有無も確認必要。
*   **Q**: 当日の延長は可能ですか？
    *   **A**: スケジュールによりますが、原則対応可能です。
*   **Q**: 支払い方法は？
    *   **A**: (現金/振込/その他 決済方法を記載)

## 5. Contact Page (`contact/index.astro`)
入力負荷を下げ、離脱を防ぐ。

### 5-1. Form
*   **Input**:
    *   お名前 (Text, Required)
    *   メールアドレス (Email, Required)
    *   電話番号 (Tel, Optional)
    *   お問い合わせ種別 (Select: 事前面談希望 / シッティング依頼 / その他)
    *   メッセージ (Textarea)
*   **Submit Button**: 「送信する」 (Loading表示あり)

### 5-2. Thanks Page
*   **Message**: 「お問い合わせありがとうございます。改めてご連絡いたします。」
*   **Action**: 「トップへ戻る」
