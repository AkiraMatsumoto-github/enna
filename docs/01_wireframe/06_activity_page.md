## 2. Specification

*   **Design Intent**:
    *   **Focus**: 料金よりも「体験価値」を前面に押し出す。写真は大きく、子どもの楽しそうな表情や作品のクオリティを見せる。
    *   **Emotional**: 「賢くなりそう」「記念になりそう」という親心に訴求。
*   **Data Source**:
    *   **Content**: Static (素材やメニューが増えたらmicroCMS化も検討だが、初期は固定でOK)
