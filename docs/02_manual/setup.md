# Project Walkthrough: enna

## 1. 成果物概要
「ママと赤ちゃんを包み込む、プロフェッショナルの温もり」をコンセプトにした、ベビーシッターサービス「enna」のWebサイトを構築しました。
Astro + TailwindCSS + microCMS を採用し、高速かつ運用しやすい構成になっています。

### 実装済みページ
*   **Top Page (`/`)**: ヒーローイメージ、特徴紹介、**microCMS連携ニュース**、お客様の声、FAQ、CTA。
*   **Profile (`/profile`)**: プロフィール写真、メッセージ、経歴タイムライン、資格一覧。
*   **Service & Price (`/service`)**: 料金プラン（ベーシック/オプション）、アクティビティへの導線。
*   **Activity (`/activity`)**: 独自の知育体験（小麦粉粘土、手形アート）の詳細紹介。
*   **Contact (`/contact`)**: 公式LINE誘導（推奨）、お問い合わせフォームUI。
*   **News Detail (`/news/[id]`)**: microCMSから取得した記事詳細ページ。

### コンポーネント・機能
*   **Responsive Header**: スマホではハンバーガーメニュー、PCではヘッダーリンク。スクロール追従。
*   **SEO & OGP**: 全ページに適切なMetaタグ、OGP設定（シェア用画像含む）を実装済み。
*   **Design System**: TailwindCSS v4 テーマ機能により、ブランドカラー・フォントを一元管理。

## 2. 検証手順

### ローカルでの動作確認
以下のコマンドで開発サーバーを起動し、ブラウザで確認してください。

```bash
npm run dev
```
→ http://localhost:4321 にアクセス

**チェックポイント:**
1.  **スマホ表示**: ブラウザの幅を狭めて（または開発者ツールでiPhoneのエミュレーション）、レイアウトが崩れていないか。
    *   ハンバーガーメニューが開閉するか。
    *   文字サイズが読みやすいか。
2.  **microCMS連携**: Topページの「お知らせ」欄に、テスト投稿した記事が表示されているか。クリックして詳細ページに遷移できるか。
3.  **リンク遷移**: 各メニュー、ボタンから正しいページに飛べるか。

## 3. Vercelへのデプロイ手順

本番環境（Vercel）へのデプロイ手順です。

1.  **GitHubへプッシュ**:
    現在のコードをGitHubリポジトリにプッシュしてください。

2.  **Vercelでプロジェクト作成**:
    *   Vercel Dashboardで「Add New Project」
    *   GitHubリポジトリ（enna）をインポート

3.  **環境変数の設定 (重要)**:
    "Environment Variables" の設定画面で、`.env` に書いた内容を登録してください。
    
    *   `MICROCMS_SERVICE_DOMAIN`: (あなたのサービスID)
    *   `MICROCMS_API_KEY`: (あなたのAPIキー)

4.  **Deploy**:
    設定後、「Deploy」をクリックすれば完了です。
    以降はGitHubにプッシュするたびに自動でデプロイされます。
